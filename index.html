NFT Metadata Scanner

About This Tool

NFT Metadata Scanner validates NFT metadata URLs, scans for security threats using VirusTotal, and safely previews metadata and media. It supports Enjin Blockchain, ERC-721, and ERC-1155 metadata standards.

  • HTTPS and IPFS URL validation
  • VirusTotal malware scanning integration
  • Metadata standard detection and parsing
  • Safe media preview with CSP protection
  • Client-side only — your data never leaves your browser
